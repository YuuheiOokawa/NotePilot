# 03. 画面遷移図

```mermaid
flowchart TD
    HOME[S-01 ホーム /] --> IDEAS[S-02 テーマ提案 /ideas]
    HOME --> LIST[S-04 記事一覧 /articles]
    HOME --> NEW[S-03 記事作成 /articles/new]
    HOME --> REVIEW[S-07 承認待ち一覧 /review]
    HOME --> REV[S-09 売上メモ /revenues]
    HOME --> SET[S-10 設定 /settings]

    IDEAS -- "このテーマで記事作成" --> NEW
    NEW -- "生成→下書き保存" --> DETAIL[S-05 記事詳細 /articles/id]
    LIST --> DETAIL

    DETAIL -- "有料設定" --> SALES[S-06 有料記事設定 /articles/id/sales]
    DETAIL -- "確認に出す (draft→review)" --> REVIEW
    REVIEW -- "承認 (review→approved)" --> DETAIL
    REVIEW -- "差し戻し (review→draft)" --> DETAIL
    DETAIL -- "コピー用プレビュー (approved以降)" --> PREVIEW[S-08 プレビュー /articles/id/preview]
    PREVIEW -- "コピー実行 (approved→copied)" --> PREVIEW
    DETAIL -- "投稿済みにする (copied→posted)" --> DETAIL
    DETAIL -- "売上を記録" --> REV
    SALES --> DETAIL
```

## ステータス遷移（ユーザー操作のみ太字）

```
idea ──記事生成──> draft ──確認に出す──> review ──**承認**──> approved ──コピー──> copied ──**投稿済み記録**──> posted
                     ^                      │
                     └──────差し戻し─────────┘
任意のステータス ──> archived（アーカイブ） / archived ──> draft（復元）
```
