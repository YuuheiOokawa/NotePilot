# 06. コンポーネント設計

## ディレクトリ構成

```
src/
├── app/
│   ├── layout.tsx            # 共通レイアウト（BottomNav・PWAメタ）
│   ├── globals.css
│   ├── page.tsx              # S-01 ホーム
│   ├── ideas/page.tsx        # S-02 テーマ提案
│   ├── articles/
│   │   ├── page.tsx          # S-04 記事一覧
│   │   ├── new/page.tsx      # S-03 記事作成
│   │   └── [id]/
│   │       ├── page.tsx      # S-05 記事詳細
│   │       ├── sales/page.tsx    # S-06 有料記事設定
│   │       └── preview/page.tsx  # S-08 コピー用プレビュー
│   ├── review/page.tsx       # S-07 承認待ち一覧
│   ├── revenues/page.tsx     # S-09 売上メモ
│   ├── settings/page.tsx     # S-10 設定
│   └── api/**/route.ts       # API（05_API設計参照）
├── components/
│   ├── BottomNav.tsx         # 下部ナビ（5タブ）
│   ├── Header.tsx            # 画面タイトル＋戻る
│   ├── StatusBadge.tsx       # ステータスバッジ（色分け）
│   ├── ArticleCard.tsx       # 記事一覧カード
│   └── CopyButton.tsx        # クリップボードコピー（コピー済み表示付き）
└── lib/
    ├── prisma.ts             # PrismaClient シングルトン
    ├── user.ts               # 既定ユーザー取得/作成
    ├── workflow.ts           # ステータス遷移表・ガード・ラベル
    ├── types.ts              # 共有型（ThemeSuggestion, GeneratedArticle, SalesPlan等）
    └── ai/
        ├── index.ts          # プロバイダ選択（env AI_PROVIDER）
        ├── mock.ts           # モック実装（既定）
        └── anthropic.ts      # Claude API実装
```

## 主要コンポーネント

| コンポーネント | Props | 役割 |
|---|---|---|
| BottomNav | なし（usePathnameで活性判定） | 全画面共通ナビ |
| Header | title, backHref? | ページヘッダー |
| StatusBadge | status | 7ステータスの色分け表示 |
| ArticleCard | article | タイトル・種別・ステータス・日付 |
| CopyButton | text, label | navigator.clipboardでコピー＋onCopied通知 |

## 状態管理

- グローバル状態ライブラリは使わず、各ページで `fetch` + `useState/useEffect`
- 更新系は実行後に再フェッチ
